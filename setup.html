itch.io Plugin – Setup

GOG Galaxy 2.1+ · 64-bit · One-time setup

⚠  
One-time setup required. The plugin requires the itch.io Desktop App as well as a personal API token, which must be generated once and saved in the existing credentials.json inside the plugin folder.
1
Download & install the itch.io Desktop App
The plugin reads your installed games from the local database of the itch.io client.

Open this URL in your regular browser (Firefox, Chrome, Edge) to download the app:
https://itch.io/app 📋 Copy Link
Install the client and launch it once – this will create the required database.
2
Log in to the itch.io client
Log in to the itch.io desktop client with your account.

Important: The database is only created after logging in, located at:
%APPDATA%\itch\db\butler.db
3
Generate API token
Open this URL in your regular browser (Firefox, Chrome, Edge) – not in the GOG window:

https://itch.io/user/oauth?client_id=3821cecdd58ae1a920be15f6aa479f7e&scope=profile&response_type=token&redirect_uri=http%3A%2F%2F127.0.0.1%3A7157%2Fgogg2itchintegration 📋 Copy Link

Log in to itch.io and click on "Authorize GOG Galaxy Integration".

The browser will then show an error page – this is normal. Look at the address bar. It will show:

http://127.0.0.1:7157/gogg2itchintegration#access_token=YOUR_TOKEN_HERE

Copy the entire token value after the = sign.
4
Edit credentials.json
Open the existing credentials.json file in the plugin folder using a text editor (e.g., Notepad):

Path
%LOCALAPPDATA%\GOG.com\Galaxy\plugins\installed\itch_2df02142-4d8a-4a4b-9b6e-c3a0bc62f93b\credentials.json 📋 Copy Path

Paste your copied token there and save the file:

{
  "access_token": "YOUR_TOKEN_HERE"
}
5
Start & connect GOG Galaxy
Restart GOG Galaxy. Go to Settings → Integrations → itch.io and click Connect.

The plugin automatically reads the token from credentials.json – no login window, no Cloudflare check.
6
Games appear in GOG Galaxy
After successfully connecting, your purchased itch.io games will be displayed under the itch.io tab.

Installed games (downloaded via the itch.io client) will additionally appear under "Installed" and can be launched directly from GOG Galaxy.

The plugin automatically checks every minute to see if new games have been installed or uninstalled.
⛔ Token Security: Your access_token grants access to your itch.io profile. Never share credentials.json with others or upload it to the internet.
🔄 Token expired? If the plugin no longer connects, repeat steps 3 and 4 with a new token.
